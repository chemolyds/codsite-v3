# SChO '23

---

SChO is our mock IChO featuring three difficulties to choose from.
Each difficulty contains a different number of questions and has a differing time to take the exam.
With this, each competitor will get roughly the same general experience.
The scores can be found [here](https://docs.google.com/spreadsheets/d/1Poh-z74ZNpZIuPwk0PZg0CqcXPCxQXcjP5ifvqedeZQ/edit?usp=sharing).
