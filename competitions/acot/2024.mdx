# ACOT '24

---

The Autumn Chemistry Open Tournament is an annual tournament that tests participants' speed and accuracy.
The 2024 competition features two rounds for each tier.
The scores can be found [here](https://docs.google.com/spreadsheets/d/1PX6G_wzYQ0Wtm7hRjY1Ry-lmMAqekxrJOtWdQ_ydz5s/edit?usp=sharing).
