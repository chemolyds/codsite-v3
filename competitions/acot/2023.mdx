# ACOT '23

---

The Autumn Chemistry Open Tournament is an annual tournament that tests participants' speed and accuracy.
The 2023 competition features two rounds for each tier.
The scores can be found [here](https://docs.google.com/spreadsheets/d/16zHD_Juqo1GxhXXLPkUGCVkeETC_OcWiGpU91tjXYWA/edit?usp=sharing).
